**Snake Game using Tkinter**
A classic Snake Game built using Python and Tkinter.
This project demonstrates GUI development, event handling, and game logic implementation using Python.

**Features**
Start Game Option
Restart Game Functionality
Multiple Difficulty Levels
Random Food Generation
Collision Detection (Wall & Self Collision)
Live Score Display

**Tech Stack**
Language: Python 3.x
GUI Library: Tkinter

**How to run**
python main.py
